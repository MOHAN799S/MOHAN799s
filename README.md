## 🚀 Key Project

### 🏛️ Civic Connect — AI-Based Public Grievance Analysis System

> *End-to-end intelligent grievance platform with real-world applicability and published research backing.*

| | |
|---|---|
| 🎯 **Scope** | Designed and deployed a full-stack platform for grievance submission and resolution |
| 🤖 **AI Pipeline** | Fine-tuned BERT & IndicBERT · Urgency detection · Sentiment classification |
| 🔍 **Explainability** | SHAP-based model transparency (X-PE module) |
| ⚖️ **Fairness** | GFAS fairness auditing to reduce prediction bias across regions & languages |
| 🌐 **Stack** | Next.js · Node.js · MongoDB · Docker · Hugging Face · Cloudinary |

**Performance Metrics:**

| Metric | Value |
|---|---|
| Classification Accuracy | 85–92% |
| Inference Time | ~0.4 seconds |
| Concurrent Users | 100+ supported |
| Fairness Score | 84 / 100 |

**Research Publication:**

> *Multimodal AI Priorities & Analysis Portal in Citizen Services*
> Published in **IJSRA Vol.18, Issue 3, 2026**
